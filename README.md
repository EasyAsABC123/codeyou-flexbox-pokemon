# Pokemon!!

hello students! let's go over the pokemon code and running it locally.

## git clone

```bash
git clone git@github.com:EasyAsABC123/codeyou-flexbox-pokemon.git
```

Change into the directory;

```bash
cd codeyou-flexbox-pokemon
cd src
```

## npm install

```bash

npm install
```

## Run the code

With nodemon:

```bash
npm run dev
```

With just node:

```bash
node server.js
```
